# Express API Boilerplate
Express API - MVC Design Pattern

### Folder Structure
```
.root
  +-- controllers
  |   +-- UserController.js
  +-- models
  |   +-- UserModel.js
  +-- routes
  |   +-- api.js
  +-- server.js
```

### Setup

#### Install Depencies
```
  npm install
```

#### Start Nodemon Server
```
  npm run start
```
